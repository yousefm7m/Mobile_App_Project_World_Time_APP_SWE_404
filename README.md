# 🌍 World Time App

A Flutter app that shows the real current time for cities around the world. Pick a city, see the time instantly, and every search gets saved to a database so you can look back at your history.

## 🚀 What the app does

- Shows the **current real time** for any city — fetched live from an API  
- Background automatically changes between **day 🌞 and night 🌙**  
- Saves every search into **MongoDB Atlas**  
- View all previous searches in the **history page**  
- Fast **search bar** to find cities instantly

## 🛠 Built with

- **Flutter** — UI framework  
- **Dart** — programming language  
- **timeapi.io** — API for world time  
- **mongo_dart** — MongoDB connection  
- **flutter_spinkit** — loading animations  
- **intl** — date & time formatting

## ✨ Features

- Real-time API data  
- Dynamic UI based on time  
- Clean and modern design  
- Persistent search history  
- Fast and responsive
